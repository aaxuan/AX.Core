基于 .Net standard 2.0.3 的通用框架类库。

原则：
🎖 开箱即用，极少代码配置
🎖 尽量减少第三方依赖
🎖 先注重功能，再注重性能
🎖 尽量覆盖 web/客户端 双端支持

⚠
依赖包 System.ComponentModel.Annotations >= 4.7.0
依赖包 Newtonsoft.Json >= 12.0.3

⚠
使用本类库请一定使用全局异常拦截，基础代码中极少使用 try，大多数参数不合法和意外情况会直接抛出异常。
可对异常类型进行判断限制本类库异常抛出。

⚠
DataBase 设计原则：
1.底层实现依赖于dapper/自实现两种模式（对外统一接口无区别
2.不支持主键自增
3.DataBaseSchema 的能力范围仅仅是从各种数据库元表结构数据到通用表结构数据。依赖于此通用结构可做任何与数据库结构有关开发。与DataBase模块无交集，可单独使用。

实验内容：

= DataBase

已完成内容：

√ CommonModel	提供一些基础的通用实体或框架实体。
√ Encryption	提供加密方法
√ Extension		提供扩展方法
√ Helper		提供帮助类
√ Net			提供邮件，网络请求等帮助方法
√ Reflection	提供简单的反射帮助方法。暂无改进计划。
√ RunLog		提供简易的日志系统。目前支持控制台日志和文件日志。

已完成并很满意内容：
√ Cache			提供了统一管理全局内存缓存的能力。工厂类可以显示全部缓存注册名称，类型，创建时间，数量等。
√ Config		提供读取 json 配置文件的能力。
